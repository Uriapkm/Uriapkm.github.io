Control de 25 bombas para síntesis automática

Cómo Trabajo

Mi proceso de diseño → fabricación → prueba → iteración

1. Descubrimiento

Entiendo el problema real del investigador: ¿qué necesita medir? ¿qué limitaciones tiene el equipamiento actual? ¿cuál es el presupuesto y timeline?

2. Diseño

Modelado CAD paramétrico en FreeCAD/OpenSCAD, simulaciones básicas si son necesarias, y revisión iterativa con el usuario final.

3. Fabricación

Impresión 3D (FDM/resina), mecanizado CNC cuando se requiere precisión, ensamblaje de componentes electrónicos y programación embebida.

4. Validación

Pruebas en condiciones reales de laboratorio, ajustes finos, documentación completa y formación al equipo investigador.
