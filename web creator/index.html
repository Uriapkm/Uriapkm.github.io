Flujo de Trabajo: Crea tu Web con el Prompt Generado

Una vez que hayas generado tu prompt, sigue estos sencillos pasos para crear y descargar tu propia página web:

1. Copia el Prompt

Haz clic en el botón "Copiar Prompt" para guardar el texto generado en tu portapapeles.

2. Elige tu IA de Desarrollo Web

Accede a una Inteligencia Artificial capaz de generar código web. Algunas opciones populares incluyen:

  • **Gemini (como esta IA):** Puedes pegarlo directamente en nuestro chat.
  • **ChatGPT (OpenAI):** Pega el prompt en la interfaz de chat.
  • **Copilot (Microsoft):** Pégalo en tu entorno de desarrollo si estás utilizando VS Code.
  • **Otras IAs especializadas en código:** Hay muchas otras disponibles o en desarrollo.

3. Pega y Ejecuta el Prompt

Pega el prompt copiado en la interfaz de la IA y envíalo. La IA comenzará a generar el código HTML, CSS y JavaScript de tu web.

4. Pega el Código Generado aquí

Una vez que la IA te dé el código, pégalo en los apartados correspondientes de abajo. **¡Importante!** Pega cada sección de código en su propia casilla (HTML en HTML, CSS en CSS, y JS en JS).

4.1. Añade tus Imágenes (Opcional)

Si tu diseño web incluye imágenes, puedes seleccionarlas aquí para que se incluyan automáticamente en el archivo ZIP. La IA debería haberte dado las rutas de las imágenes en el HTML (ej. `src="images/mi-imagen.jpg"`).

Puedes seleccionar múltiples imágenes a la vez.

**¡Importante!** Si la IA te sugiere una carpeta para las imágenes (ej. `images/`), asegúrate de usar esa ruta en tu código HTML para que las imágenes se muestren correctamente al descomprimir.

4.2. Añade tu Logo (Opcional)

Si marcaste la opción de "Incluir Logo" en el prompt, aquí puedes subir el archivo `logo.png` que se incluirá en el ZIP.

Asegúrate de que el archivo sea `logo.png` para que se muestre correctamente en la web generada por la IA.

5. Descarga tu Web

Cuando tengas el código y las imágenes listos, haz clic en el botón para descargar tu web en un archivo ZIP.

6. Abre tu Web Localmente

Descomprime el archivo ZIP descargado. Dentro encontrarás `index.html`, `style.css`, `script.js` y, si los añadiste, una carpeta `images/` con tus fotos. Para ver tu página web, simplemente haz doble clic en el archivo `index.html` en tu carpeta. Se abrirá en tu navegador predeterminado.

7. Personaliza y Refina

El código generado es una base. Puedes abrir los archivos `.html`, `.css` y `.js` con un editor de texto simple (como Notepad++, VS Code, Sublime Text) y modificar:

  • **Textos:** Cambia los títulos, párrafos y contenido para adaptarlos a tu información real.
  • **Imágenes:** Sustituye las imágenes de ejemplo por las tuyas propias (asegúrate de que las rutas en el HTML/CSS sean correctas).
  • **Estilos:** Ajusta colores, fuentes, espaciados y responsive para un diseño perfecto.
  • **Funcionalidad:** Modifica o extiende el JavaScript si necesitas interacciones más complejas.

Recuerda que cada vez que hagas un cambio en los archivos, debes guardar y recargar la página en tu navegador para ver las actualizaciones.

8. ¡Publica tu Web! (Opcional)

Una vez que estés satisfecho con tu web, puedes publicarla para que sea accesible online. Esto generalmente implica:

  • **Contratar un Hosting:** Un servicio que almacena tus archivos web.
  • **Obtener un Dominio:** La dirección de tu web (ej. `tudominio.com`).
  • **Subir los Archivos:** Usar un cliente FTP o el panel de control del hosting para subir tu carpeta (`index.html`, `style.css`, `script.js`, etc.).

Si necesitas ayuda con la publicación, puedes generar otro prompt para la IA preguntándole "cómo publicar mi página web creada con HTML, CSS y JS".
