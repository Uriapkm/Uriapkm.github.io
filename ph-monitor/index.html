Arduino Setup

Download and upload the Arduino code to your device to interface with the pH sensor.

Upload Instructions

  1. Download and install the Arduino IDE
  2. Connect your Arduino board to your computer via USB
  3. Open the downloaded file in the Arduino IDE
  4. Select your board type from Tools > Board (e.g., Arduino Uno)
  5. Select the correct port from Tools > Port
  6. Click the Upload button (right arrow) to upload the code
  7. Once uploaded, the Arduino will start sending pH data via serial port
Note: Ensure your pH sensor is connected to pin A0 on the Arduino with 5V reference voltage.

Why This App Exists: The pH Probe That Drove Me Crazy

TL;DR: The old pH monitoring method was needlessly complex. I built this browser-based app so everyone could use the sensor without IT support.

The Problem: Too Complicated to Share

We have an Arduino pH monitoring system at CIC Energigune, originally brought from MIT. While the hardware works fine for tracking pH in our redox flow batteries, the software workflow was a technical nightmare.

The original process involved manual code changes, command-line scripts, and constantly crashing serial ports. It was so convoluted that I knew my lab mates would never use it, making me the de facto IT support for simple pH measurements.

The Old Way vs. The New Way

The Old Way: Editing Arduino code, recompiling, re-flashing firmware, fighting with PowerShell scripts just to open COM ports.
The New Way (This App): Flash the Arduino once, then handle all calibration and data logging in the browser with simple clicks. No command line, no re-flashing.

Key Features

  • Zero-install - Runs entirely in your browser
  • Real-time monitoring - Live pH readings with beautiful charts
  • Simple calibration - No code editing required
  • Data export - CSV and Excel formats with statistics
  • Cross-platform - Works on Windows, Mac, Linux
  • Dark mode - Easy on the eyes during long experiments

Mission accomplished: Now anyone in the lab can measure pH without calling for help every five minutes!
