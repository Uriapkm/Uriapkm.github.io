Instructions for Uploading to Arduino

  1. Download and install the Arduino IDE.
  2. Connect your Arduino board to your computer via USB.
  3. Download the phmonitor.ino file by clicking the link above.
  4. Open the downloaded phmonitor.ino file in the Arduino IDE.
  5. In the Arduino IDE, select your board type (e.g., Arduino Uno) from Tools > Board.
  6. Select the correct port from Tools > Port (the port your Arduino is connected to).
  7. Click the Upload button (right arrow) in the Arduino IDE to upload the code to your board.
  8. Once uploaded, the Arduino will start sending pH data to this web application via the serial port.

Note: Ensure your pH sensor is connected to pin A0 on the Arduino, and use a 5V reference voltage.

Why This App Exists: The pH Probe That Drove Me Crazy

We have an Arduino pH monitoring system here at CIC Energigune, which was originally brought over by a girl who spent a few months researching at MIT. It's used for tracking the pH in our redox flow batteries, and the hardware is fine. The method for actually using it? It was a technical nightmare.

The Problem: Too Complicated to Share

When I was first shown how to get a reading and calibrate the probe, I immediately knew this was broken. The process was so ridiculously convoluted—involving manual software changes, running command-line scripts, and dealing with constantly crashing serial ports—that I honestly stopped paying attention halfway through.

It wasn't that I couldn't do it, but I knew my lab mates wouldn't touch it. I was facing a future where I'd be the only person capable of running a simple pH measurement. I decided I wasn't going to be stuck being the lab's dedicated IT support.

The Old Way vs. The New Way

The Old Way: Every time we needed a precise calibration, it required editing the Arduino code itself, recompiling, and re-flashing the firmware onto the board. Plus, connecting to the computer meant fighting with PowerShell scripts just to open and close the COM port. It was pure chaos.

The New Way (This App): I built this zero-install web page using the Web Serial API because it eliminates 99% of that pain. You flash the Arduino once, and after that, all calibration and data logging are handled right here in the browser with a couple of clicks. No more command line, no more re-flashing. Just chemistry.

TL;DR:

The old pH monitoring method was needlessly complex, required coding skills, and forced re-flashes for calibration. I built this simple, browser-based app so everyone could use the sensor without calling me every five minutes.
